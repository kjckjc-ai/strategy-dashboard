# Trust Dashboard v8

### Key Features:
- No filters or iframes
- Schools grouped by phase of education
- Clean layout with profile cards
- Trust summary stats (KPIs)
- Export full data as CSV or Excel

## How to Use
1. Place `National datasheeet.csv` in the same folder.
2. Run:
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
